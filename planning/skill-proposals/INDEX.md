## Proposal Process

1. **Propose**: Use `/aget-propose-skill` or create entry manually
2. **Review**: Assess fit, scope, dependencies
3. **Approve/Reject**: Decision with rationale
4. **Implement**: Create skill per SKILL.template.md
5. **Deploy**: Add to .claude/skills/
6. **Validate**: Run dependency validation (L586)
